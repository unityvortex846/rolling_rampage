# Rolling Rampage

A browser-based aura-collection game. Roll to collect auras of increasing rarity — the rarer the aura, the cooler the visual effect and the bigger your stat boost.

## Play

[**Play on GitHub Pages →**](https://jenrum3ry.github.io/rolling_rampage/)

## Features

- **8 rarity tiers**: Common → Uncommon → Rare → Epic → Legendary → Mythic → Divine → Phantom
- **Luck multipliers**: Potions temporarily boost your odds of landing rarer auras
- **Aura effects**: Particle glow system that scales up dramatically with rarity
- **Tutorial**: Onboarding flow with a free Beginner Potion (10,000× luck for 1 roll)
- **Gauntlets**: Scaffolded system (coming soon) — require specific auras to unlock rewards
- **Persistent state**: Progress saved automatically via localStorage

## Development
